# 📌 Sticky Header & Go To Top Button

## 📖 About

This project demonstrates how to create a sticky header that stays at the top of the page while scrolling and a "Go To Top" button that helps users quickly return to the top. It uses HTML for the page structure and an external CSS file for styling, positioning, and layout, making website navigation simple and user-friendly.

## ✨ Features

* 📌 Sticky Header using `position: sticky`
* ⬆️ Go To Top Button
* 🎨 External CSS Styling
* 🌐 Simple Navigation Menu
* 📱 Responsive Layout
* 💻 Beginner-Friendly Code

## 🚀 Purpose

The purpose of this project is to learn how to use the CSS `position: sticky` property and create a fixed navigation feature with a "Go To Top" button to improve user experience and website navigation.
